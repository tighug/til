# Prisma

- Prisma Client
- Prisma Migrate
- Prisma Studio

## Installation

=== "npm"

    ```bash
    npm install -D @prisma/cli
    npm install -S @prisma/client
    ```

=== "yarn"

    ```bash
    yarn add -D @prisma/cli
    yarn add @prisma/client
    ```
