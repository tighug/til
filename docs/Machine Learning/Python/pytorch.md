# PyTorch

PyTorch は Python のオープンソースな機械学習ライブラリです。

## Installation

```bash
poetry add torch numpy
```

## Usage

```py
import torch
```

## API

### Tensors

```py
tensor.add(x, y) # x + y
tensor.sub(x, y) # x - y
tensor.mul(x, y) # x * y
tensor.div(x, y) # x / y
```
