# Redux Toolkit

Redux Toolkit とは、**Redux を使いやすくするためのツールキット**です。

## Installation

```bash
yarn add @reduxjs/toolkit
```

## Usage

以下の API が使用できる。

- `configureStore()`
- ~~`createReducer()`~~
- ~~`createAction()`~~
- `createSlice()` : 上二つの代わりに使う
- `createAsyncThunk`
- `createEntityAdapter`
- `createSelector`
