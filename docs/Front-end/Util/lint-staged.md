# lint-staged

## Installation

```bash
yarn add -D lint-staged
```

## Configuration

`package.json`あるいは`.lintstagedrc`ファイルに記述します。

```json title="package.json"
{
    "lint-staged": {
        "*": "your-cmd"
    }
}
```

```json title=".lintstagedrc"
{
    "*": "your-cmd"
}
```
