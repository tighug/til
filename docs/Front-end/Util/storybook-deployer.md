# Storybook Deployer

## Installation

=== "Yarn"

    ```bash
    yarn add -D @storybook/storybook-deployer
    ```

## Usage

=== "Yarn"

    ```bash
    yarn storybook-to-ghpages   # To GitHub Pages
    yarn storybook-to-aws-s3    # To AWS S3
    ```
