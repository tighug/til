# ts-node

ts-node とは、**Node.js で TypeScript を実行したい時に、JavaScript にトランスパイルすること無く、そのまま実行できるツール**です。

## Installation

=== "NPM"

    ```bash
    npm install -D ts-node
    ```

=== "Yarn"

    ```bash
    yarn add -D ts-node
    ```

## Usage

=== "NPM"

    ```bash
    npx ts-node [filename.ts]
    ```

=== "Yarn"

    ```bash
    yarn run ts-node [filename.ts]
    ```
