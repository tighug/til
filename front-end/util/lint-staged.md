# lint-staged

## Installation

```bash
yarn add -D lint-staged
```

## Configuration

`package.json`あるいは`lintstagedrc`ファイルに記述します。

```json
// package.json
{
  "lint-staged": {
    "*": "your-cmd"
  }
}
```

```json
{
  "*": "your-cmd"
}
```
