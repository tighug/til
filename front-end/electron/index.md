# Electron

Electron は、**クロスプラットフォームのデスクトップアプリを開発するためのフレームワーク**です。Chronium と Node.js を利用しているため、ブラウザアプリと同じ技術でデスクトップアプリが開発可能です。

## Installation

```bash
yarn add -D electron
```

## CLI

```bash
electron [main file]  # Run Electron App
```
