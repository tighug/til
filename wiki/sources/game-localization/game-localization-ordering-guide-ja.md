## 要約

SunFlareによるゲームローカライズの翻訳依頼先の選び方と依頼方法の解説。フリーランス vs 翻訳会社の比較、依頼時に必要な準備物を紹介。

## 依頼先の選択肢

1. **フリーランス翻訳者**: 求人サイト・SNSで募集。タイトル愛好家を起用可能。コスト安だが、品質チェック工程がなく個人の力量に依存
2. **オンライン翻訳サービス**: Nitro（Alconost）、Gengo等。翻訳者を自分で探す手間なし。文字単価計算が一般的
3. **翻訳会社**: プロジェクト管理・セキュリティ・複数チェック工程付き。コスト高だが品質担保が手厚い
4. **ゲーム特化翻訳会社**: ゲーム専門知識を持つPM・翻訳者が担当。利用規約・プロモーション動画等もまとめて対応可能

## フリーランス vs 翻訳会社の比較

| 項目 | フリーランス | 翻訳会社 |
|------|------------|---------|
| コスト | 安い傾向 | やや高い |
| 品質チェック | なし（個人依存） | 翻訳後チェック・校正あり |
| コミュニケーション | 直接やり取り、細かい指示可 | PM経由 |
| スケーラビリティ | 大規模・多言語は困難 | 対応力高い |
| 翻訳者選定 | 自社で評価が必要 | 事前にテスト済み翻訳者をアサイン |

## 依頼時に準備すべきもの

- 仕様書・企画書などの参考資料
- キャラクターの相関図
- スタイルガイド

## 翻訳会社への依頼フロー

1. 上記資料を準備して見積り・交渉
2. 翻訳会社が翻訳者選定・スケジュール設定
3. ファミリアライズ（ゲーム理解）→ インゲームテキスト翻訳 → LQA → 納品
4. 必要に応じて用語集・スタイルガイドの作成も翻訳会社が担当可能
